Script ASAR.py solves the problem stated in 'project1.pdf' file: A* search on Airline Scheduling and Routing.
Input and output file formats are described in the referenced pdf file.

To run the script do:
> $ ./ASAR.py PATH_TO_INPUT_FILE

folder 'examples' has some possible input files.

This script is implemented upon the aima package. See aima/README.md for more information on their package usage.
