Script ASAR.py solves the problem stated in 'project2.pdf' file: Bayesina Networks and probability propagation
Input file format is described in the referenced pdf file.

To run the script do:
> $ ./Bayes.py PATH_TO_INPUT_FILE

folder 'public_tests' has some possible input files.

This script is implemented upon the aima package. See aima/README.md for more information on their package usage.
